About This Project

This project demonstrates a complete end-to-end DevOps pipeline built by Bahavani Gajanand. It showcases how a simple web application can be containerized, orchestrated, and continuously deployed using industry-standard tools. From writing code to pushing to GitHub — automated pipelines handle building Docker images, running tests, and deploying to a Kubernetes cluster running on AWS EC2. Infrastructure is provisioned using Terraform with state stored remotely in AWS S3 with DynamoDB locking for production-grade reliability.
